# Personal Portfolio Website

Welcome to the repository for my personal portfolio website!

This project serves as a central place to showcase my experience, technical projects, and interests in IT and cybersecurity. It's also been a great opportunity to strengthen my front-end development skills while creating a website that reflects my personality and professional growth.

🌐 **Live Website:**  
https://stevedorm.github.io/PortfolioWebsite/

---

## About the Project

Rather than relying on a website builder, I wanted to create my portfolio from the ground up using HTML, CSS, and JavaScript. Building it myself has given me a much better understanding of responsive web design, layout techniques, and user experience.

The site continues to evolve as I gain new experience, complete projects, and learn new technologies.

---

## Current Features

- Responsive design for desktop and mobile devices
- Project showcase
- Skills and technologies overview
- Resume page
- Contact information
- Modern UI with smooth navigation
- Custom styling and animations

## Why I Built It

This website is more than just an online résumé—it's a project in itself.

I wanted a place where I could:

- Showcase projects I'm proud of
- Document my technical growth
- Experiment with web development
- Give recruiters and employers an easy way to learn more about me

As my skills continue to grow, so will this website.
